#  Calculator Project

This project is a simple and responsive **Calculator** built using **HTML, CSS, and JavaScript**.  
It supports all basic arithmetic operations, real-time input handling, clear/AC functions, and keyboard support.

---

##  Features

###  Basic Arithmetic  
- Addition (+)  
- Subtraction (−)  
- Multiplication (×)  
- Division (÷)

###  Clean User Interface  
- Button-based keypad  
- Digital display screen  
- Smooth hover effects  

###  Functional Buttons  
- **AC** → Clear all  
- **C** → Delete last character  
- **=** → Show result  
- **%** → Modulus  

###  Real-Time Display  
- Shows every number and operator instantly  
- Evaluates safely using JavaScript  

###  Keyboard Support (Bonus)  
Use your keyboard to type numbers and operators:  
- 0–9 keys  
- +, -, *, /  
- Enter → equals  
- Backspace → delete  
- Escape → clear

##  How to Run the Project

### **On Mobile (Acode / HTML Viewer):**
1. Create a new folder  
2. Add the 3 files:  
   - index.html  
   - style.css  
   - script.js  
3. Open `index.html` → Live Preview  
4. Calculator is ready to use

### **On Desktop:**
1. Download or clone the repository  
2. Open `index.html` in any browser  
3. No server required
